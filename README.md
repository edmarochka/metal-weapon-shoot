Metal Weapon: Мережевий тактичний шутер на Python

Metal Weapon — це динамічна багатокористувацька гра в жанрі Top-down Shooter, розроблена для дуельних боїв у реальному часі. Проєкт поєднує в собі класичну аркадну механіку з сучасними програмними рішеннями на базі бібліотеки Pygame для візуалізації та фреймворку CustomTkinter для створення інтерфейсу користувача.

В основі гри лежить клієнт-серверна архітектура, реалізована за допомогою низькорівневих мережевих сокетів (Socket). Сервер не лише координує підключення, а й генерує ігровий простір, передаючи клієнту структуру мапи у форматі JSON. Це забезпечує ідентичність ігрового поля для обох опонентів. Обмін даними відбувається безперервно: гравці передають свої координати, стан здоров'я та масиви випущених куль, що дозволяє досягти плавної синхронізації ігрового процесу.

Ігрова механіка включає систему зіткнень з перешкодами, де стіни слугують стратегічним захистом, блокуючи постріли. Реалізовано систему перезарядки (Reload Time) та візуалізацію стану здоров'я за допомогою динамічних HP-барів. Особливу увагу приділено стабільності з'єднання: код містить алгоритми для обробки фрагментованих пакетів даних, що запобігає вильотам гри при нестабільному інтернет-з'єднанні.

Гра має гнучку систему рендерингу: вона підтримує використання кастомних спрайтів (наприклад, у стилі легендарної серії Metal Gear), але при їх відсутності автоматично переходить на процедурну генерацію графіки, що робить продукт відмовостійким. Metal Weapon демонструє ефективне використання мови Python для створення повноцінних мережевих додатків, поєднуючи складну логіку бекенду з інтерактивним графічним інтерфейсом.
